# HuggingFace Model Training

This repository contains exercises and examples for learning Hugging Face model training.

## Resources

- Lesson video: [Bilibili lesson](https://www.bilibili.com/video/BV1ta4y1g7bq/?spm_id_from=333.1007.top_right_bar_window_history.content.click&vd_source=25935d75dbd538261892c3a32d02909c)

## Prerequisites

- Python 3.9
- Use a virtual environment for package isolation

## Getting Started

1. Create and activate a virtual environment.
2. Install required packages.
3. Follow the lesson and experiment with model training.
